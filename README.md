# Gen AI Context Engineering

This repository contains exercises and labs focused on context engineering for Large Language Models.

## 📁 Repository Structure 

### [Chapter 1: Fundamentals](README.md)
All initial exercises are organized within the `chp1` directory:

* **[Approach Switchboard](./chp1/approach-switchboard/docs)**: Strategies for routing prompts.
* **[Context Skeleton Studio](./chp1/context-skeleton-studio/docs)**: Building robust prompt frameworks.
* **[Context Tetris](./chp1/context-tetris/docs)**: Managing token limits and information density.
* **[Dynamic Facts Firewall](./chp1/dynamic-facts-firewall/docs)**: Techniques for grounding and hallucination prevention.
* **[Enterprise Scenario](./chp1/enterprise-scenario/docs)**: Large-scale context implementation.
* **[Failure Mode Clinic](./chp1/failure-mode-clinic/docs)**: Debugging and edge-case handling.
* **[Iteration Lab](./chp1/iteration-lab/docs)**: Rapid prototyping and prompt refinement.
* **[Memory Mixer](./chp1/memory-mixer/docs)**: Long-term context and state management.
* **[RAG Snippet Surgeon](./chp1/rag-snippet-surgeon/docs)**: Optimizing retrieval-augmented generation.
