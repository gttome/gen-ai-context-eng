# Gen AI Context Engineering

This repository contains exercises and labs focused on context engineering for Large Language Models.

## 📁 Repository Structure 

### [Chapter 1: Fundamentals](./chp1)
All initial exercises are organized within the `chp1` directory:

* **[Approach Switchboard](./chp1/docs/approach-switchboard)**: Strategies for routing prompts.
* **[Context Skeleton Studio](./chp1/docs/context-skeleton-studio)**: Building robust prompt frameworks.
* **[Context Tetris](./chp1/docs/context-tetris)**: Managing token limits and information density.
* **[Dynamic Facts Firewall](./chp1/docs/dynamic-facts-firewall)**: Techniques for grounding and hallucination prevention.
* **[Enterprise Scenario](./chp1/docs/enterprise-scenario)**: Large-scale context implementation.
* **[Failure Mode Clinic](./chp1/docs/failure-mode-clinic)**: Debugging and edge-case handling.
* **[Iteration Lab](./chp1/docs/iteration-lab)**: Rapid prototyping and prompt refinement.
* **[Memory Mixer](./chp1/docs/memory-mixer)**: Long-term context and state management.
* **[RAG Snippet Surgeon](./chp1/docs/rag-snippet-surgeon)**: Optimizing retrieval-augmented generation.
