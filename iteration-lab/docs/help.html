Help

This app is a sandbox for disciplined context iteration: Design → Test → Evaluate → Adjust, changing one thing at a time.

Workflow

  • Design: build your context blocks. Use Context Health to watch ordering warnings + token bloat.
  • Test: copy the context + question into your LLM and paste outputs back.
  • Evaluate: score outputs on grounded / relevant / consistent / structured / safe. Each test shows a delta vs the previous iteration.
  • Adjust: create the next iteration and lock editing to one block. Use the Fix palette suggestions if a criterion is low.

Trend + regressions

  • Trend chart appears after you score at least one iteration.
  • Regression watch warns on both checklist averages and per-test drops.

Resume / Start New

If a saved run exists in this browser, the app will ask whether to resume it or start a new run. Use “Export Saved Run” to download your last run JSON before clearing.
