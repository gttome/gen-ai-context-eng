# Mobile QA Scorecard — v0.5.0

- [ ] Phone portrait: no horizontal scroll for primary UI flows
- [ ] Touch targets: buttons/inputs are tap-friendly
- [ ] Keyboard: focus visible; tab order usable
- [ ] Contrast: text readable in light/dark themes
- [ ] Orientation: phone landscape remains usable
- [ ] Tablet: layout scales without overlap
- [ ] Desktop: responsive resize works (narrow window)
- [ ] Performance: no obvious lag with ~50 tests and ~10 iterations
- [ ] Compare Diff: diff output readable on small screens (wraps, no clipping)
- [ ] Scenario Builder modal: usable on phone (scrollable, no trapped content)
