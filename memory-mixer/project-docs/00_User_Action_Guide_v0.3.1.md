# User Action Guide — Memory Mixer — Iteration 2 — v0.3.1

## What this package is
A **hotfix build** that resolves a load-time JavaScript syntax error reported in v0.3.0.

A **mobile-first** static HTML5 app (GitHub Pages compatible) for practicing:
- Rolling Summary
- Pinned Facts
- Retrieval Memory (now with **search + tags**)
- Drift + continuity discipline
- **Conflict reconciliation** (Pinned vs Summary)
- **Run portability** via Export/Import JSON

## Run locally (Windows)
1. Unzip the package.
2. Open the `docs/` folder.
3. Double-click `start-server.bat`.
4. Your browser opens the app.

## How to use (fast path)
1. Click **New Run** (or **Resume** if you have a saved run).
2. Work each turn: capture notes into **Roll up / Pin / Retrieve**.
3. If the **Drift Alarm** appears, pin the missing critical decision(s).
4. Use **History** to restore or diff snapshots.
5. Use **Retrieval Search** + **Tag filter**; click **Tags** on any retrieval item.
6. If **Conflicts** > 0, click **Review** and choose:
   - Replace summary (make summary match pinned), or
   - Update pinned (make pinned match summary).

## Portability (move a run between devices)
- Click **Export Run JSON** to download a complete run file.
- Click **Import Run JSON** to restore it later (even on another device/browser).

## Reset behavior
- If idle: **Reset** clears any saved run in this browser.
- If running: **Reset** clears the current run’s memory workspace.
