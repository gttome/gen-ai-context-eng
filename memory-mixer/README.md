# Memory Mixer — Iteration 2 Hotfix (v0.3.1)

Memory Mixer is a mobile-first, static HTML5 learning app that teaches **multi-turn memory management**:
- Rolling summary (continuity without dumping history)
- Pinned facts (durable decisions/facts that must not drift)
- Retrieval memory (supporting notes you can store for later)

## What’s new in Iteration 2
- Retrieval Memory **search + tag filter**
- Per-retrieval **Tags** editing
- **Conflicts** detector (Pinned Facts vs Rolling Summary)
- **Export/Import Run JSON** (portable across devices/browsers)
- Accessibility polish (skip link, improved modal actions)

## Run locally (Windows)
1. Open the `docs/` folder
2. Double-click `start-server.bat`

## Deploy to GitHub Pages
- Publish from `/docs`
- Verify Environment pill shows **GitHub Pages**
