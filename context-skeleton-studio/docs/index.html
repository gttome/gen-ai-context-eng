    How to Use (Quick)

    1. Pick a workflow (or load a preset/fixture).
    2. Edit the blocks in plain language.
    3. Click Validate and fix warnings.
    4. Use the length hint to trim non-essential text.
    5. Copy/download text or export a JSON snapshot.
