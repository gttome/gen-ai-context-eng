Gen AI Context Engineering

This repository contains exercises and labs focused on context engineering for Large Language Models.

📁 Repository Structure

Chapter 1: Fundamentals

All initial exercises are organized within the chp1 directory:

  • Approach Switchboard: Strategies for routing prompts.
  • Context Skeleton Studio: Building robust prompt frameworks.
  • Context Tetris: Managing token limits and information density.
  • Dynamic Facts Firewall: Techniques for grounding and hallucination prevention.
  • Enterprise Scenario: Large-scale context implementation.
  • Failure Mode Clinic: Debugging and edge-case handling.
  • Iteration Lab: Rapid prototyping and prompt refinement.
  • Memory Mixer: Long-term context and state management.
  • RAG Snippet Surgeon: Optimizing retrieval-augmented generation.
