# User Action Guide (v0.3.0)

## Run locally (Windows)
1. Unzip the release
2. Open docs/
3. Double-click start-server.bat
4. Open http://localhost:8000/

## Use the app
- Pick a scenario → sort cards → Check
- Change Time Warp date to see staleness/contradictions
- Optional: enable Challenge mode before starting
- Open Worksheet to print a paper exercise
- Use Export blocks to copy prompt blocks

## If something breaks
- Open DevTools Console and copy the first error line
- Tell me what you clicked and the scenario name
