# Dynamic Facts Firewall (v0.3.0) — Iteration 2

A mobile-first HTML5 mini-app (GitHub Pages compatible) that trains context-engineering learners to keep **Dynamic Facts** separate from **Role** and **Rules**—so prompts don’t rot when values change.

## What’s new in Iteration 2
- Misconception callouts + mini-quiz
- Challenge mode (timer + moves)
- Time Warp contradiction signals
- Printable Worksheet page (`worksheet.html`)

## Run locally (Windows)
1. Unzip the release.
2. Open `docs/`.
3. Double-click `start-server.bat`.
4. Your browser should open to `http://localhost:8000/`.

## Deploy to GitHub Pages
Publish the `/docs` folder as your Pages site:
- Settings → Pages → Build and deployment → Deploy from a branch → Folder: `/docs`

## What’s new in Iteration 1
- Bin micro-lessons (in-app)
- Hints after repeated mistakes + “Auto-place” teaching action
- Stronger Time Warp staleness signals
- Local progress log (best score + recent runs)
- Deck Builder page (`docs/builder.html`) for creating/importing custom decks
