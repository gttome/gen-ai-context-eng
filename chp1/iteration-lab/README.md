# Iteration Lab (v0.5.0)

A mobile-first, GitHub Pages–ready HTML5 app that teaches disciplined context iteration:
**Design → Test → Evaluate → Compare → Adjust** (change one thing at a time).

## What’s new in v0.5.0 (Iteration 4 — P0 features)
- **Scenario Builder (local-only):** create your own scenarios (baseline blocks + test set) inside the app.
- **Block-level line diffs (Compare):** a lightweight line-by-line diff view per changed block.
- **Export Bundle (.json):** one download that includes the run JSON, the current iteration report (Markdown), iteration log, templates, scenarios, and event log.
- **Trend guardrails:** trend chart now **breaks segments** and warns when the **test set changes**, so you don’t misread apples-to-oranges trends.

## Run locally (Windows)
1. Open `docs/`
2. Double-click `start-server.bat`
3. Open the shown localhost URL in your browser

## Deploy to GitHub Pages
Publish the `docs/` folder as a GitHub Pages site (static hosting).
