# Release Notes — v0.3.5
**Date:** 2026-02-27

## Fixed / Improved
- App now starts **empty** (no default task selected).
- Added **Reset App** (header) for a true fresh start (clears task selection + pack + results).
- Renamed **Reset** to **Reset Pack** (keeps selected task; clears window + results).
- Disabled **Score Pack** + **Apply Quick Fixes** until a task is selected (with friendly warning toast if clicked).
- Block Queue displays an instructional message until a task is selected.
- Cache-busting updated to `?v=v0.3.5`.
