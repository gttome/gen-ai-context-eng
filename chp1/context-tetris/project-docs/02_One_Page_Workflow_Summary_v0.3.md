# One-Page Workflow Summary — v0.3.1

## Loop (single-builder + MobAI agents)
1. **Choose scope** (one iteration target)
2. **Update PRD + backlog**
3. **Implement** (complete files)
4. **QA gate** (Mobile QA scorecard)
5. **Package zip** (docs + manifest + handoff + agent report)
6. **Deploy to GitHub Pages** (publish `/docs`)
7. **Inspect/adapt** (next iteration)

## Release rules
- Mobile-first is release-critical.
- Static-site compatible (GitHub Pages).
- Zip-first; silent screen output.
- Always write/update `project-docs/28_MobAI_Agent_Report_vX.md`.
