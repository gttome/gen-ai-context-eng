# Agile Iteration Process + Agent Team — v0.3.1

## Cadence
- One small target per iteration (1–2 hours of work)
- Ship zip package each time
- Evaluate on phone + desktop before calling it “done”

## Minimal MobAI roster
- Product Lead — scope + success criteria
- Architect — structure + constraints (Windows/GitHub Pages)
- Builder — implement complete files
- UI/UX — mobile-first interaction + accessibility
- QA — scorecard gates + smoke tests
- Security — client-only safe patterns
- Release Steward — packaging, manifest, handoff, agent report

## Required artifact every time
- `project-docs/28_MobAI_Agent_Report_vX.md` (current request only)
