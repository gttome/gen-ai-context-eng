# Context Tetris (v0.3.5)

A mobile-first microgame that teaches **Context Engineering fundamentals** by having you pack context blocks into a finite token window.

## Local run (Windows)
1. Open `docs/`
2. Double-click `start-server.bat`
3. Open `http://localhost:8000/`

> Note: the app loads JSON files via `fetch()`, so you must use a local server (not `file://`).

## GitHub Pages
- Publish the `/docs` folder.
- The app is 100% static (HTML/CSS/JS + JSON).


## Reset
- **Reset Pack** clears the current Context Window + results.
- **Reset App** clears the selected task + pack + results (fresh start).
