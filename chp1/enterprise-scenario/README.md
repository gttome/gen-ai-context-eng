# Enterprise Scenario Arcade (v0.4.0)

A **mobile-first**, static HTML5 app for practicing **Context Engineering** with enterprise-style scenarios (Support / HR / Ops).

## What’s new in v0.4.0
- Added **10 new business-domain scenarios** across Support / HR / Ops (19 total scenarios).
- Scenarios cover: SLA escalation, IT access/security, invoice dispute, accommodation request, termination pressure, travel/expense exception, supplier delay, quality CAPA, sole-source procurement, and quarter-end variance.


## Run locally (Windows)
1. Unzip the package
2. Run: `docs\start-server.bat`
3. Open: `http://localhost:8000/`

## Deploy to GitHub Pages
- Publish the `docs/` folder as the Pages root (static site).
