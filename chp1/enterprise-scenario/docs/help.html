Help

Enterprise Scenario Arcade is a mobile-first practice tool for Context Engineering. It does not call an LLM—your goal is to assemble a better context package and evaluate quality.

Quick start

  1. Pick a station (Support, HR, or Ops).
  2. Edit the Context Skeleton blocks (Role, Rules, Dynamic Facts, Excerpts, Memory, Output Format, Request).
  3. Click Generate Response to get a template-based draft (you can edit it).
  4. Click Score to run checks and see Coach Alerts.

Tabs

  • Response: draft response + Audit Mode (line support tags).
  • Score: pass/fail checks, Coach Alerts, and Run History.
  • X-Ray: block-level health check (missing blocks, placeholders, format drift).

Exports

  • Copy Context Package / Download Context .txt: paste into ChatGPT / Gemini / Claude.
  • Download Run Report: one file containing context, response, score, alerts, and audit tags.

Reset vs Clear History

  • Reset clears your current workbench state.
  • Clear History removes saved run history entries.
