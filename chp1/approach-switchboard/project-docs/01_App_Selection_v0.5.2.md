## Selected Chapter 1 app
**Approach Switchboard** — an interactive decision console that helps learners choose between basic prompting, context engineering, retrieval (RAG), and fine-tuning based on Chapter 1.

## Why this app
- Directly operationalizes Chapter 1’s:
  - “Prompting vs context engineering vs fine-tuning (decision guide)”
  - “Signals that you need more than basic prompting”
  - “Context window & tokens (working memory model)”
  - “Input-quality principle” (generic/wrong/inconsistent output diagnosis)
- Fast, safe MVP: static, no backend, no secrets, GitHub Pages friendly.
