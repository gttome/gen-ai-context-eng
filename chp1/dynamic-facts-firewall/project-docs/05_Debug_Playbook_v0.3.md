# Debug Playbook (v0.3.0)

## Common issues
- **Blank page**: ensure you are using http://localhost:8000 (not file://)
- **JS error**: open DevTools Console; copy the first error line
- **Buttons don’t work**: confirm JS files load (Network tab) and paths are relative
- **Clipboard blocked**: browser may restrict clipboard; use fallback prompt or different browser

## How to report a bug
1. What you clicked (step-by-step)
2. Scenario name
3. Browser + device
4. Console error text (first error)
5. Screenshot if UI rendering issue
