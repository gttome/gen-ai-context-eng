# Mobile QA Scorecard (v0.3.0) — Strict Release Gate (Pass/Fail)

## Required devices / viewports
- Phone portrait (baseline)
- Phone landscape
- Tablet
- Desktop/laptop
- Narrow desktop window (resized)

## Gates (must PASS)
1. **No horizontal scroll** for normal content on phone portrait
2. Tap targets comfortably usable (≥44px)
3. Keyboard navigation works; visible focus states
4. No hover-only controls required
5. Time Warp date control usable on mobile
6. Check/Reset/Export/Worksheet all usable on mobile
7. No console errors in core flows
8. Content readable without zoom tricks (zoom allowed; not disabled)
9. Worksheet page prints cleanly (header/actions hidden in print)
