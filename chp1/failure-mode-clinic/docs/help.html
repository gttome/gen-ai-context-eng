Quick start
  1. Go to Library and load a case.
  2. In Clinic, choose Symptom → Cause → Fix, then Score.
  3. Tap Generate fix to produce a “fixed context” and review the diff.
  4. Use Builder (or the Wizard) to create your own cases in My Pack.
  5. Use Drill mode for fast practice rounds.
Tip: On Windows, run start-server.bat from the docs folder for reliable local behavior.
