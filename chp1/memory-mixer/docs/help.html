Help

Quick guide to the core controls and Iteration 2 features.

New Run / Resume
App starts idle. Click New Run, or Resume a saved run.
Retrieval Search + Tags
Use Search + Tag filter. Click “Tags” on a retrieval item to label it.
Conflicts
If the Rolling Summary disagrees with a Pinned Fact, the Conflicts score shows a count. Use Review to reconcile.
Export / Import Run JSON
Export a full run to a JSON file and import it later on any device/browser.

Quick tips

  • Rolling Summary: short roll-up of what matters so far.
  • Pinned Facts: durable decisions + facts you must not forget.
  • Retrieval Memory: extra details you might search and re-inject later.

Iteration 1 tools

  • Required hints: toggle on/off to see what must be pinned for the current turn.
  • Replay panel: toggle on/off to jump across turns and review what you captured per turn.
  • Snapshots: use Save Snapshot, then open History to restore or compare A vs B with a diff.
