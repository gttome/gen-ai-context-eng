# Development Guardrails — v0.3.1

## Static site rules (GitHub Pages)
- No secrets in client code
- Relative paths only
- Works without a build step
- Optional APIs must be HTTPS + CORS safe

## Mobile-first rules
- Phone portrait first
- Tap-friendly controls
- No hover-only interactions
- Visible focus + keyboard support

## Release discipline
- Zip-first, silent screen output
- Update manifest + handoff + agent report every package
